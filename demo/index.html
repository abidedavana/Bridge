The differentiator · security

It refused the trap.

Bridge runs untrusted repo code and applies model‑written diffs. A hostile repo hid an instruction in its build output telling the agent to add a shell‑out. Even when the model obeyed, the mechanical policy gate rejected the diff before it reached git — pinned by an end‑to‑end test.
